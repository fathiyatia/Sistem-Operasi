BAB 9

# cd
mengganti direktori

# pwd
mencetak nama direktori saat ini

# ls
menampilkan daftar isi direktori

# touch
meng-update waktu akses dan modifikasi suati file, jika file belm ada maka touch akan membuat file kosong

# cp
menyalin file dan direktori

# mv
memindahkan (mengganti nama) file

# rm
menghapus file atau direktori

# mkdir
membuat direktori

# rmdir
menghapus dorektori kosong
